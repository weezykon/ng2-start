# ng2-start
angular 2 quickstart
![ng2-start](https://cloud.githubusercontent.com/assets/586490/25270467/b98e3780-2678-11e7-8925-efa5968bc38d.JPG)

#npm install

To view/ Launch Browser
#npm start
